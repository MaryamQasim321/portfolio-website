Real-Time Chat Application

This is a real-time chat application built using WebSockets, allowing users to communicate instantly with one another. The application provides a simple interface where users can enter their names and send messages to a chatroom that broadcasts the message to all connected users. It demonstrates core functionality for real-time messaging, including username setting, message broadcasting, and handling multiple users simultaneously.

Key Features:

Set Username:

Users can input their name upon connecting to the chat.

Real-Time Messaging:

Messages are broadcast in real-time to all users in the chatroom using WebSockets.

User-Friendly Interface:

Simple and clean UI for an easy chat experience.

Responsive Design:

Works seamlessly across desktop and mobile devices.

Technologies Used:

Frontend:

HTML, CSS, JavaScript

Backend:

Node.js with the ws WebSocket library

Real-Time Communication:

Powered by WebSocket for instant message transmission

Project Highlights:

Implemented WebSocket protocol to enable efficient real-time communication between clients and the server. Designed the frontend with clean and minimalistic UI to focus on seamless chat interactions. Built server-side logic to handle multiple connected users, manage message broadcasts, and handle disconnections gracefully.
